## Critérios para Atualização

Atualizar este documento sempre que ocorrer um dos eventos abaixo:

* conclusão de um módulo;
* início de um novo módulo;
* conclusão de um volume;
* mudança de prioridade;
* alteração significativa no planejamento;
* redefinição do próximo marco.

Não registrar pequenas alterações editoriais ou correções de texto.

---

## Como Retomar o Projeto

Ao iniciar uma nova sessão de trabalho:

1. Ler os documentos de governança.
2. Consultar este documento.
3. Identificar o próximo marco.
4. Localizar o último arquivo produzido.
5. Continuar exatamente desse ponto.

Nunca reiniciar módulos concluídos.

Nunca alterar a estrutura do projeto sem autorização explícita.
