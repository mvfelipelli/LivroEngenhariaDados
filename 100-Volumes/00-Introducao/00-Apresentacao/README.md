# Bem-vindo à Academia de Engenharia de Dados

Seja bem-vindo à **Academia de Engenharia de Dados**.

Esta formação foi desenvolvida para preparar profissionais capazes de projetar, construir, implantar, monitorar e evoluir plataformas modernas de dados, utilizando tecnologias amplamente empregadas pelo mercado.

Ao contrário de treinamentos focados apenas em ferramentas específicas, esta Academia tem como objetivo desenvolver uma visão completa da Engenharia de Dados, desde os fundamentos até arquiteturas distribuídas de nível corporativo.

Durante a formação, construiremos uma plataforma de dados completa, semelhante às utilizadas em empresas de médio e grande porte.

---

# Objetivos desta Etapa

Ao concluir esta etapa você será capaz de:

- compreender o papel da Engenharia de Dados dentro das organizações;
- diferenciar Engenharia de Dados, Ciência de Dados, Analytics e Business Intelligence;
- conhecer a arquitetura geral que será construída durante a formação;
- preparar corretamente o ambiente de laboratório;
- compreender a metodologia utilizada em toda a Academia;
- identificar a trilha de aprendizagem e as competências esperadas em cada módulo.

# Pré-requisitos

Recomenda-se que o aluno possua:

- conhecimentos básicos de informática;
- familiaridade com sistemas operacionais;
- interesse em programação;
- vontade de aprender através da prática.

Não é necessário conhecimento prévio em Engenharia de Dados.

---

# Convenções Utilizadas

Ao longo da formação serão utilizados os seguintes padrões:

- exemplos em Linux e Windows quando necessário;
- SQL ANSI sempre que possível;
- PostgreSQL como banco de referência;
- Python como linguagem principal;
- Spark para processamento distribuído;
- Docker como plataforma de laboratório.
