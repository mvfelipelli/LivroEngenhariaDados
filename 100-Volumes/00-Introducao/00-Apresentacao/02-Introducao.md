# Introdução

A Academia de Engenharia de Dados é uma base de conhecimento progressiva que combina livro, curso, wiki e laboratório. Seu objetivo é desenvolver raciocínio técnico durável antes de apresentar ferramentas específicas.

```mermaid
flowchart LR
    C["Conceitos"] --> F["Fundamentos"]
    F --> I["Funcionamento interno"]
    I --> P["Prática"]
    P --> O["Operação"]
    O --> E["Evolução"]
```

O leitor começa pela natureza dos dados e avança até arquiteturas, plataformas e operação. Cada volume reutiliza decisões anteriores e amplia o projeto da DataRetail S.A.

> [!note]
> A trilha é sequencial, mas o Vault permite consulta transversal por Wikilinks.
