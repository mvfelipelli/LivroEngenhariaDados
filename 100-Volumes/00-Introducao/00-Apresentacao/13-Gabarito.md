# Gabarito

1. Deve mencionar fundamentos, progressão prática e construção de competências verificáveis.
2. Estão na raiz de `100-Volumes/00-Introducao`.
3. Estruturado possui diretórios/controles; concluído possui conteúdo obrigatório produzido e validado.
4. SQL consulta, modelagem define semântica, Python automatiza e Spark distribui processamento.
5. Exemplos: exercício explicado, laboratório validado e decisão arquitetural registrada.
6. Deve reservar leitura, prática, revisão e registro de dúvidas.
7. Segredos, dados pessoais, credenciais, informações contratuais e dados proprietários.
8. O README deve declarar contexto, metas mensuráveis, cronograma, evidências e limitações.
