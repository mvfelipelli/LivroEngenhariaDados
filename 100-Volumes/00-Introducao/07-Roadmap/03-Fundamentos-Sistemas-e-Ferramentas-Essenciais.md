# Fundamentos, Sistemas e Ferramentas Essenciais

Os primeiros volumes constroem vocabulário e controle do ambiente. Fundamentos apresentam ciclo de vida, bancos, modelagem, ETL e pipelines. Linux desenvolve compreensão de processos, arquivos e automação. Git oferece histórico e colaboração. SQL desenvolve raciocínio relacional.

Essas competências são multiplicadoras: troubleshooting de Spark depende de sistema operacional; modelagem depende de granularidade; revisão de pipeline depende de Git; quase toda plataforma oferece SQL.

Evidências recomendadas:

- explicar um fluxo de dados e seus contratos;
- automatizar tarefa segura no shell;
- trabalhar com branches e resolver conflito;
- escrever consultas com joins e validar cardinalidade.

Pular essa base transfere dificuldade para módulos mais caros e distribuídos.
