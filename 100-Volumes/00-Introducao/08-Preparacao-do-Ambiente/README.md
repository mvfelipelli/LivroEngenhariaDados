# Preparação do Ambiente

Este módulo converte os componentes apresentados em [[100-Volumes/00-Introducao/05-Ambiente-da-Academia/README|Ambiente da Academia]] em uma instalação verificável. O resultado esperado é uma estação capaz de versionar conteúdo, executar Python, iniciar serviços em containers, acessar PostgreSQL e diagnosticar falhas básicas.
