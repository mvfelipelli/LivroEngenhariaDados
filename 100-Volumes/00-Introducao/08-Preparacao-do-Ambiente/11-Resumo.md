# Resumo — Preparação do Ambiente

- Faça preflight antes da instalação.
- Registre versões e decisões.
- Isole Python e alinhe Java com Spark.
- Declare serviços com containers e preserve dados em volumes.
- Restrinja PostgreSQL à rede necessária.
- Valide cada camada com testes de fumaça.
- Nunca versione segredos.
- Documente diagnóstico e recuperação.
