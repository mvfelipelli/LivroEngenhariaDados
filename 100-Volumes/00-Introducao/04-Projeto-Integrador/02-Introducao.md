# Introdução

Exercícios isolados ensinam mecanismos; um projeto contínuo ensina consequências. Uma decisão de modelagem afeta SQL, processamento, qualidade, custo e consumo. Por isso a Academia reutiliza a mesma empresa e seus produtos.

```mermaid
flowchart LR
    P["Problema"] --> M1["Marco mínimo"]
    M1 --> E["Evidência"]
    E --> M2["Nova capacidade"]
    M2 --> R["Revisão arquitetural"]
    R --> E
```

O projeto cresce por incrementos executáveis. Nenhuma etapa depende de uma plataforma corporativa paga; dados sintéticos e ambientes locais permitem reproduzir os fundamentos.
