# Gabarito

1. Deve registrar valores, espaço livre, virtualização e limitações.
2. Interface, interpretador e programa de linha de comando.
3. UTF-8, fim de linha, formatação, terminal, Git e extensões necessárias.
4. Segredos, `.env`, ambientes, caches, logs, dados grandes e builds.
5. Executor; isolamento; dependência distribuída; resolução reproduzível.
6. Imagem cria instância; container executa; volume persiste; rede conecta.
7. Identificar processo, necessidade, porta configurável, impacto e encerramento seguro.
8. Mínimo: editor/Git/Python; completo adiciona containers, Java e serviços conforme trilha.
