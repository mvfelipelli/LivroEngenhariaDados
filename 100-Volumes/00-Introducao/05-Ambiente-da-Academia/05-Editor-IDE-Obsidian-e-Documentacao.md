# Editor, IDE, Obsidian e Documentação

Editor trabalha com texto; IDE acrescenta execução, depuração e análise; Obsidian transforma Markdown em Vault navegável. Uma ferramenta pode cumprir múltiplas funções, mas arquivos continuam sendo a fonte portátil.

Configure encoding UTF-8, final de linha consistente, formatação e integração com Git. Extensões devem resolver uma necessidade conhecida e não esconder comandos fundamentais.

O Obsidian usa Wikilinks, backlinks, callouts, Mermaid e frontmatter. Configurações de interface pessoais podem mudar sem representar conteúdo editorial; por isso arquivos de workspace não devem ser tratados como entrega.

Documente no mesmo fluxo da implementação. README desatualizado é um defeito operacional, não apenas estético.
