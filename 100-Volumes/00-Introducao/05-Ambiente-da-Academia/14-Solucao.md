# Solução — Inventário Reproduzível do Ambiente

```markdown
# Inventário do Ambiente

## Plataforma

- Sistema: Windows 11 / Linux x86_64
- Shell: PowerShell 7 / Bash
- CPU: 8 núcleos
- RAM: 16 GB
- Disco livre para laboratórios: 60 GB
- Virtualização: habilitada

## Ferramentas

- Git: versão registrada
- Python: versão registrada
- Java: versão registrada ou indisponível
- Container runtime: versão registrada ou indisponível
- Editor/Obsidian: instalados

## Limitações

- memória permite até três serviços pesados simultâneos;
- dados e imagens devem ser limpos periodicamente;
- segredos e identificadores foram omitidos.
```

Adapte o modelo ao host real e atualize quando o ambiente mudar.
