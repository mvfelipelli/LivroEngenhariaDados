# Objetivos

Ao concluir este módulo, você será capaz de:

- explicar o caminho entre código-fonte, bytecode e execução;
- selecionar uma versão Python compatível com o projeto;
- criar ambientes virtuais isolados;
- declarar e fixar dependências de modo reproduzível;
- distinguir REPL, script, módulo, pacote e comando;
- organizar configuração sem expor segredos;
- usar formatação, lint, testes e tipos como feedback automatizado;
- entregar uma CLI mínima com códigos de saída verificáveis.

> [!tip]
> O objetivo não é decorar comandos, mas tornar a execução repetível em notebooks, terminais, CI e produção.
