# Módulo 01 — Dados

> [!quote]
> "Dados são a matéria-prima de toda decisão baseada em informação."

---

# Objetivo

Neste módulo estudaremos o conceito mais importante de toda a Engenharia de Dados.

Tudo começa com os dados.

Antes de aprender bancos de dados, Spark, SQL ou Lakehouse, precisamos compreender profundamente:

- o que são dados;
- quais tipos de dados existem;
- como são produzidos;
- como evoluem;
- como devem ser tratados.

---

# O que você aprenderá

Ao concluir este módulo você será capaz de:

- definir dados;
- diferenciar dado, informação e conhecimento;
- identificar tipos de dados;
- compreender qualidade dos dados;
- entender metadados;
- reconhecer problemas comuns.
