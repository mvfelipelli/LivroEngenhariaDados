# Volume 01 — Fundamentos da Engenharia de Dados

> [!quote]
> "Toda tecnologia muda. Bons fundamentos permanecem por décadas."

---

# Objetivo

Este volume estabelece toda a base conceitual necessária para compreender Engenharia de Dados.

Ao concluir este volume você será capaz de compreender:

- dados
- informação
- bancos de dados
- pipelines
- ETL
- ELT
- arquiteturas
- processamento
- armazenamento
- qualidade
- governança

Todo o restante da Academia depende deste volume.

---

# Pré-requisitos

É recomendado ter concluído:

- [[100-Volumes/00-Introducao/README]]

---

# Objetivos de Aprendizagem

Ao final deste volume você será capaz de:

- explicar o ciclo de vida dos dados;
- diferenciar OLTP e OLAP;
- compreender ETL e ELT;
- projetar pipelines simples;
- identificar arquiteturas modernas;
- compreender Batch e Streaming;
- entender a importância da qualidade dos dados.
