# Referências

## Documentação oficial

- Git Project. *Git Reference Manual*.
- CHACON, Scott; STRAUB, Ben. *Pro Git*, segunda edição.
- Git Project. `gitglossary`, `gitrevisions`, `gittutorial` e `gitworkflows`.
- Git Project. Documentação de `gitcore-tutorial`, protocol v2 e formatos de repositório.

## Tópicos recomendados

- `git help revisions` para seleção de commits;
- `git help reset` para as três árvores;
- `git help merge` e `git help rebase` para integração;
- `git help credentials` e `git help config` para autenticação e política;
- Git LFS para ponteiros de arquivos grandes.

> [!tip]
> Consulte a documentação da versão instalada. Defaults como branch inicial, estratégia de pull e algoritmo de objeto podem variar.
