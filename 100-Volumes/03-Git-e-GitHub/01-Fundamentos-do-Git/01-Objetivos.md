# Objetivos

Ao concluir este módulo, você será capaz de:

- explicar versionamento distribuído e snapshots;
- distinguir working tree, index, repositório local e remoto;
- interpretar blobs, trees, commits, tags, refs e HEAD;
- criar commits coesos e inspecionar diferenças e histórico;
- trabalhar com branches, merges e conflitos;
- sincronizar repositórios com fetch, pull e push;
- desfazer mudanças na área correta sem perda acidental;
- recuperar commits com reflog e referências;
- escolher workflow, política de qualidade e segurança;
- executar um fluxo Git completo e reproduzível.

O módulo é concluído ao executar [[14-Laboratorio|o laboratório]] duas vezes com as mesmas invariantes.
