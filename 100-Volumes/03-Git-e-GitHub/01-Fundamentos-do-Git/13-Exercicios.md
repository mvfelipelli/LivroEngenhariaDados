# Exercícios

## Revisão

1. Diferencie blob, tree, commit e ref.
2. Explique working tree, index e HEAD.
3. Compare branch e tag.
4. Diferencie fetch, pull e push.

## Interpretação

5. Arquivo possui mudanças staged e não staged. Quais diffs inspecionar?
6. Commit foi feito em detached HEAD. Como preservá-lo?
7. Push foi rejeitado por non-fast-forward. Proponha fluxo seguro.
8. Um segredo está no commit anterior. Quais ações são necessárias?

## Aplicação

9. Crie três commits coesos para migration, pipeline e teste.
10. Produza e resolva conflito de merge.
11. Recupere uma branch removida usando reflog.
12. Desenhe `.gitignore` para projeto Python de dados.

## Desafios

13. Compare merge, squash e rebase em política de integração.
14. Proponha workflow para migrations de banco.
15. Execute [[14-Laboratorio]] e explique o grafo final.

Consulte [[13-Gabarito]] após registrar os comandos e justificativas.
