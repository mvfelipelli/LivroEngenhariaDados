# Objetivos

Ao concluir este módulo, você será capaz de:

- escolher modelo de colaboração e permissões;
- projetar branches curtas e estratégias de integração;
- construir pull requests revisáveis e rastreáveis;
- distinguir aprovação, checks e decisão de merge;
- usar rebase, squash e cherry-pick com consciência;
- configurar proteção, rulesets, CODEOWNERS e bypass;
- integrar issues, projetos, releases e deploys;
- reduzir riscos de Actions e supply chain;
- definir governança de repositórios para dados;
- simular colaboração distribuída no laboratório.
