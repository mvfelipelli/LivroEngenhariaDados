# Exercícios

## Revisão

1. Diferencie evento, workflow, job, step e action.
2. Compare cache, artifact e output.
3. Explique matrix, needs e concurrency.
4. Diferencie secret, variable, OIDC e environment.

## Interpretação

5. Workflow de fork usa secret de produção. Diagnostique.
6. Cada ambiente recompila o projeto. Qual risco?
7. Dois deploys simultâneos executam migrations. Proponha controles.
8. Action usa tag `@main`. Explique o risco.

## Aplicação

9. Desenhe CI para pipeline Python e SQL.
10. Defina permissões por job.
11. Modele promoção por digest com staging e produção.
12. Crie estratégia OIDC com claims restritas.

## Desafios

13. Compare runner hospedado e self-hosted.
14. Projete reusable workflow organizacional.
15. Execute [[14-Laboratorio]] e introduza dependência cíclica.

Consulte [[13-Gabarito]] após justificar as decisões.
