# Exercícios

## Revisão

1. Diferencie ameaça, vulnerabilidade, risco e controle.
2. Explique defesa em profundidade e menor privilégio.
3. Compare DAC, ACL, capabilities, LSM e seccomp.
4. Diferencie prevenção, detecção e recuperação.

## Interpretação

5. Um scanner recomenda desativar IPv6 em host que o utiliza. Como tratar?
6. Uma conta de serviço possui shell e sudo irrestrito. Proponha correções.
7. Um segredo foi removido do Git, mas continua válido. Qual ação falta?
8. Patch foi instalado, mas kernel não mudou. Diagnostique.

## Aplicação

9. Crie baseline mínimo para gateway SFTP.
10. Planeje mudança segura de SSH sem lockout.
11. Defina logs e auditoria para uso de sudo.
12. Modele exceção temporária com controle compensatório.

## Desafios

13. Crie matriz de risco para host de banco.
14. Planeje resposta a chave privada exposta.
15. Execute [[14-Laboratorio]] e introduza uma violação para validar a falha.

Consulte [[13-Gabarito]] após justificar as decisões.
