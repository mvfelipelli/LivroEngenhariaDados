# Exercícios

1. Monte `spark-submit` com dois argumentos da aplicação.
2. Escolha client ou cluster mode para job agendado.
3. Empacote módulo Python e driver JDBC sem incluir segredos.
4. Compare Standalone, YARN e Kubernetes para uma empresa existente.
5. Dimensione slots iniciais para 200 partições.
6. Defina limites de dynamic allocation para carga variável.
7. Projete canary com reconciliação monetária.
8. Desafio: escreva runbook para backlog streaming crescente.
