# Exercícios

1. Defina opções seguras para CSV com cabeçalho e registro corrompido.
2. Compare Parquet, JSON e CSV para um mart analítico.
3. Diferencie column pruning, predicate pushdown e partition pruning.
4. Projete uma leitura JDBC com limite de oito conexões.
5. Explique por que `overwrite` não basta para idempotência.
6. Escolha partição física para consultas diárias por região.
7. Proponha métricas para detectar small files.
8. Desafio: desenhe publicação atômica em armazenamento de objetos.
