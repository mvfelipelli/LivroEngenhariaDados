# Exercícios

1. Defina schema para pedido com ID, instante UTC e valor decimal.
2. Normalize UF com funções nativas e trate nulos.
3. Explique a diferença entre Row, Column e DataFrame.
4. Substitua uma UDF de maiúsculas por expressão nativa.
5. Exploda itens aninhados preservando pedidos sem itens.
6. Identifique as quatro fases exibidas por `explain("extended")`.
7. Proponha métricas para detectar casts que viraram `null`.
8. Desafio: projete evolução compatível de schema para adicionar descontos.
