# Módulo 01 — Fundamentos e Níveis de Modelagem

Modelar dados é explicitar conceitos, identidade, regras e relações para que sistemas preservem o significado do domínio. O modelo é simultaneamente ferramenta de descoberta, contrato e base para decisões técnicas.
