# Referências

## Fontes fundamentais

- CODD, E. F. *A Relational Model of Data for Large Shared Data Banks*. CACM, 1970.
- ARMSTRONG, William W. *Dependency Structures of Data Base Relationships*. 1974.
- DATE, C. J. *Database Design and Relational Theory*. O'Reilly.
- ELMASRI, Ramez; NAVATHE, Shamkant. *Fundamentals of Database Systems*. Pearson.
- MAIER, David. *The Theory of Relational Databases*. Computer Science Press.

## Documentação

- PostgreSQL Global Development Group. [Constraints](https://www.postgresql.org/docs/current/ddl-constraints.html).
- SQLite Project. [Foreign Key Support](https://www.sqlite.org/foreignkeys.html).
